# Security Policy

This is a community fork of
[clusterzx/paperless-ai](https://github.com/clusterzx/paperless-ai). For
upstream security issues, please report to the upstream maintainer.

## Supported Versions

This fork is maintained on a best-effort basis by the fork owner. Only the
latest release is supported.

| Version | Supported          |
| ------- | ------------------ |
| current (HB42 v0.9+) | :white_check_mark: |
| older   | :x:                |

## Reporting a Vulnerability

For security vulnerabilities specific to this fork, please open a private
[security advisory](https://github.com/hendrik-b42/paperless-ai/security/advisories)
on GitHub rather than a public issue.

For vulnerabilities that also exist in upstream, please report upstream
first and coordinate disclosure.
